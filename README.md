# E-Generation - 英単語学習アプリ

ブラウザベースの英検各級対応英単語学習アプリ

## 機能

- 🎯 英検5級〜1級対応
- 📱 4択クイズシステム
- 🏆 進捗管理・ステージクリア
- 👤 ユーザー管理（Admin制御）
- ☁️ クラウド同期（Firebase）
- 📊 スコア管理

## 技術スタック

- **フロントエンド**: Next.js 14, TypeScript, Tailwind CSS
- **アニメーション**: Framer Motion
- **データベース**: Firebase Firestore
- **デプロイ**: Vercel

## 開発環境での起動

```bash
npm install
npm run dev
```

## デプロイ

Vercel + GitHubで自動デプロイ

## 初期管理者アカウント

- ユーザー名: `admin`
- 管理画面でユーザーアカウントを作成可能
